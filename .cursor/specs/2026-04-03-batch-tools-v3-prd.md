# 批量小工具（v3）PRD

## 目标

新增 3 个工具页：哈希摘要（Web Crypto）、正则测试、颜色格式转换。

## 需求与 AC

| 编号 | 描述 | AC |
|---|---|---|
| V3-1 | SHA-256/384/512：对输入文本（UTF-8）计算 digest，输出 hex | 算法可切换；非法输入不崩溃；可一键复制 |
| V3-2 | 正则测试：输入 pattern/flags/文本，展示匹配列表或错误 | 无效正则显示错误；可展示 match 文本 |
| V3-3 | 颜色转换：解析 HEX/RGB/HSL，输出互转结果 | 解析失败提示；可复制 |
| V3-4 | 统一接入 `useToolVisit` 与首页入口 | 路由可访问；构建通过 |

## 非功能

- 哈希计算使用 `crypto.subtle`（HTTPS 或 localhost 下可用）。
- 正则执行在客户端，注意 ReDoS：对超长输入做长度限制提示。
